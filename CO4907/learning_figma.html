Activity 1. Getting Familiar with the Interface

  • Watch the video walkthrough to get familiar with the Figma UI and observe some of the basic features that are listed below.

  • Become familiar with the interface

    • Create a new document: To create a new document, click on the "Create New" button in the top-left corner of the Figma homepage. From here, you can choose the type of document you want to create, such as a web design, mobile app, or icon set.

    • Add layers: Layers are the building blocks of your design in Figma. To add a new layer, click on the "Layers" button on the left sidebar, and then click on the "+" button to add a new layer. You can also use the "Layer" menu in the top toolbar to add different types of layers, such as text, shapes, and images.

    • Layers → every new element you add creates a layer automatically. You can rearrange layer by drag and drop.

    • Use the toolbar: The toolbar at the top of the Figma interface contains all the tools you need to create your design. You can use the selection tool to select and move layers, the pen tool to draw freeform shapes, and the rectangle tool to create rectangles. You can also use the colour picker to change the colour of your layers, and the alignment tools to align your layers to each other.

    • Use the canvas: The canvas is where you will do most of your design work in Figma. You can use the zoom and pan tools to navigate around the canvas, and you can use the rulers and grids to help you align your layers.

    • Export your design: Once you are happy with your design, you can export it by going to the "File" menu and selecting "Export". From here, you can choose to export your design as an image, PDF, or HTML file.

Additional

Components: Components are reusable elements that you can use across different pages and frames in your document. You can create a component by selecting a layer or group of layers, and then clicking on the "Create Component" button in the top toolbar. Once you've created a component, you can edit it in one place, and the changes will be reflected across all instances of the component.

Styles: Styles allow you to define and apply consistent design elements across your document. There are different types of styles, such as text styles, color styles, and grid styles. You can create a new style by selecting a layer or group of layers, and then clicking on the "Create Style" button in the top toolbar. Once you've created a style, you can apply it to other layers by selecting them and clicking on the desired style in the "Styles" panel.

Auto-layouts: Auto-layouts allow you to create responsive designs that automatically adjust to different screen sizes. You can use auto-layouts to create flexible grids, align elements to the edges of the frame, and create responsive components. To use auto-layouts, you can use the "Auto-Layout" button in the top toolbar, and then set the constraints for each element.

Plugins: Figma has a large library of plugins that you can use to extend the functionality of the tool. You can install plugins from the Figma website and use them to perform tasks such as generating design tokens, creating wireframes, and exporting designs to code.

Collaboration: Figma allows you to collaborate with other designers in real-time and share your design with stakeholders. You can invite people to a file, give them edit access or view access, and use the commenting and prototyping tools to get feedback and test your design. You can also version control, and revert to previous versions of your design if needed.

APIs: Figma has a set of APIs that you can use to automate and integrate with other tools. With the Figma API, you can programmatically access your design data, create new designs, and even generate code snippets.

Design Systems: Design Systems are a set of design principles, guidelines, and components that are used consistently throughout a product. You can use Figma to create a design system by using symbols, styles, and auto-layouts to create a library of reusable components. This will help you to maintain consistency and efficiency across your designs.

Advanced Auto-layouts: Auto-layouts are a powerful feature of Figma that allows you to create responsive designs. You can use advanced auto-layouts to create flexible grids, align elements to the edges of the frame, and create responsive components.

Advanced prototyping: Figma has a range of prototyping tools that allow you to create interactive wireframes and mockups. You can use the advanced prototyping techniques to create animations, triggers, and micro-interactions.

Advanced use of plugins: Plugins can be a great way to extend the functionality of Figma. You can use plugins to automate repetitive tasks, generate design tokens, and export designs to code.

Advanced use of APIs: Figma's API allows you to programmatically access your design data and automate tasks. You can use the API to create custom integrations with other tools, such as creating a plugin for a specific task, or automating the export process.

Workflow Automation: You can use a combination of techniques, plugins and API to automate your workflow, create custom scripts, or use third-party tools to automate repetitive tasks, such as generating design tokens, creating wireframes, and exporting designs to code.
